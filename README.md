# Projeto Final I – Metodologia Científica

Repositório destinado ao armazenamento das atividades e versões produzidas na disciplina **Projeto Final I**, etapa inicial da construção do Trabalho de Conclusão de Curso do bacharelado em **Sistemas de Informação**.

## Finalidade do Repositório

Este espaço tem como objetivo centralizar os materiais elaborados ao longo da disciplina, especialmente os textos relacionados à definição e estruturação do anteprojeto, com base nas orientações metodológicas apresentadas em aula.
